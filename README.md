# ♻ EcoSmart Bin AI

## 🌍 Problem

People often throw wrong waste in recycling bins because they don’t understand recycling rules.

---

## 💡 Solution

EcoSmart Bin AI uses your camera to detect waste and tell you how to dispose it correctly.

---

## 🚀 Features

* 📸 Scan items using camera
* ♻ Detect Plastic, Paper, Metal, Glass
* 🔊 Voice instructions
* 🌱 Eco Points system
* 🤖 AI-powered detection (COCO-SSD)

---

## 🛠 Technologies Used

* HTML, CSS, JavaScript
* TensorFlow.js
* COCO-SSD Model

---

## ▶️ How to Run

1. Open project in browser using Live Server
2. Click **Scan Item**
3. Show object to camera
4. Get result instantly

---

## 🎯 Example

* Bottle → Plastic
* Notebook → Paper
* Can → Metal
